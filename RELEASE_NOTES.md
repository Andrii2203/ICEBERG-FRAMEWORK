# ICEBERG FRAMEWORK — RELEASE NOTES
A chronological record of all changes made to the Iceberg Framework.  
Each entry corresponds to a version defined in VERSIONING.md.

---

# 1. FORMAT
Each release entry must include:
- version number  
- release date  
- type of changes  
- summary  
- affected documents  

Example structure:
v0.2.0 — 2026-01-16
Type: Minor Release
Summary

Added Documentation Standard

Added Terminology / Glossary

Added CONTRIBUTING.md

Added VERSIONING.md

Affected Documents
DOCUMENTATION_STANDARD.md

TERMINOLOGY.md

CONTRIBUTING.md

VERSIONING.md
